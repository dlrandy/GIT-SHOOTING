git push的时候发送了什么？
所有新的commit和当前分支相关的对象

两种修改git push的默认方式
1. 要git询问分支名字在每次push的时候，直接Git push么有效果
git config --global push.default nothing
2. 只有同名的远端的时候，才可以push
git config --global push.default simple

设置编辑器
Windows需要设置编辑器的PATH变量；
git config --global core.editor notepad

设置别名
git config --global alias.br branch
git config --global alias.co checkout

git config --global alias.cm "commit -m"

git config --global alias.unstage 'reset HEAD --'

git config --global alias.undo 'reset --soft HEAD~1'

git config --global alias.last 'log -1 HEAD'

git config --global alias.difflast 'diff --cached HEAD^'

想要别名运行外部的shell命令，在别名的实体前加上！
git config --global alias.echo !echo

git config --global --unset alias.cm
