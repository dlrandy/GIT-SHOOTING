git log -p 展示所有的改动详情
git log  --decorate展示commit的所有引用详情
git log --stat 展示所有的改动统计
git log -n 展示n条数据

git log --grep='s' 查看commit信息里有s的 

git log -S't'查看code里有t的

git log -Gs\|t正则表达式匹配
-i -ignore case
-no-merges 过滤掉merge操作

git log master..feature查看它俩分支之间的commit
git log file查看某个文件的改动

git log origin/master..拿master和当前分支作比较
